# 🧠 D-Red-Bot  
### Developed by **Quick Red Tech** | Created by **Chisom Life Eke**

![Platform](https://img.shields.io/badge/platform-windows-blue?style=for-the-badge)  
![Version](https://img.shields.io/badge/version-Qv1.5.0-red?style=for-the-badge)

**D-Red-Bot** is a **smart system memory utility for Windows** developed by **Quick Red Tech**,  
engineered by **Chisom Life Eke**, the creator of the **RedMaxima** and **StackCheckMate** projects.  

It provides developers and enthusiasts with a futuristic **boot-simulation CLI** that mimics an OS startup,  
while analyzing and displaying **live memory addresses**, **file sizes**, and **system logs** — all in real time.  

🌐 **Official Website:** [https://qrtquick.github.io/D-Red-Bot/](https://qrtquick.github.io/D-Red-Bot/)  
📦 **GitHub:** [https://github.com/QRTQuick/D-Red-Bot](https://github.com/QRTQuick/D-Red-Bot)

---

## ⚙️ Key Features
- 🧠 **Predictive Memory Tracking:** Scans memory blocks and predicts address allocations dynamically  
- 💻 **Linux-Style Boot Console:** Cool boot animation logs with subsystem startup lines  
- 🪟 **Windows GUI Alerts:** Built-in MessageBox notifications for important logs  
- 💾 **File Insight Engine:** Lists files with name, size, memory reference, and creation time  
- ⚡ **Ultra Lightweight C++ Engine:** Runs entirely from the console — no dependencies  
- 🔐 **Custom CLI Interface:** Menu-driven system with options for Start, Search, Settings, and Exit  

---

## 🧰 Tech Stack
- **Language:** C / C++ (Windows)  
- **Core Engine:** Win32 API, iostream, dirent, sys/stat  
- **Compilation:** MinGW G++ / Visual Studio Developer Tools  
- **Category:** System Utility / Memory Diagnostic Tool

## 🚀 Installation & Usage
1. Clone this repository:
   ```bash
   git clone https://github.com/QRTQuick/D-Red-Bot.git
   cd D-Red-Bot

2. Compile with:

g++ log_mem_address.cpp -o D-Red-Bot.exe


3. Run:

D-Red-Bot.exe


4. Navigate the CLI menu to start scanning or monitoring.
