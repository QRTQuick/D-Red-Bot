🧠 Overview

D-Red-Bot is a futuristic Windows system utility built in C++ by Quick Red Tech (founded by Chisom Life Eke). It simulates an intelligent OS boot sequence while performing live memory and file analysis inside your console.

The tool includes a predictive address resolver, heap tracker, live file scanner, and even GUI alert integration using native Windows APIs — blending CLI + GUI experiences.

⚙️ Core Features

  • 🧩 Predictive memory tracking and visualization
  • 📂 Real-time file buffer scanning with address pointers
  • 🧠 AI-style boot simulation and subsystem logs
  • 🎛️ Settings menu with color and GUI toggle
  • 🪟 Windows-native performance — lightweight and fast

🚀 How to Run

git clone https://github.com/QRTQuick/D-Red-Bot.git
cd D-Red-Bot
g++ D-Red-Bot.cpp -o D-Red-Bot.exe
D-Red-Bot.exe

Once compiled, the boot screen will start automatically. Use the interactive menu to scan files, predict memory, and view live address tracking.
